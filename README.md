# 🚀 MyDashServ

> Fullstack business application built with React (Vite) & Spring Boot.  
> Designed to manage complex scheduling logic, multi-role workflows and secure REST communication.

---

## 🧠 Business Context

MyDashServ simulates a real-world training organization management system.

The application handles:

- Multi-role user management (Admin / Manager / Trainer / School)
- Smart trainer-session matching
- Automatic scheduling conflict detection
- Role-based workflow validation
- Secure REST API with JWT authentication

This project demonstrates the ability to design and implement structured business logic within a production-ready fullstack architecture.
